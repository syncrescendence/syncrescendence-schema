# Atrium

Cross-cutting index of institutionally significant artifacts from all salons.

Nothing is authored here. The atrium holds symlinks to artifacts that meet promotion criteria defined in `methods/artifact-templates/ARTIFACT-TEMPLATES-VIZIER.md`. An artifact earns an atrium symlink when it establishes or revises institutional architecture, carries a calibration judgment that downstream work depends on, is a ratified specification or confirmed receipt, or is explicitly promoted by the Sovereign or Vizier.

The atrium stays lean. It indexes the institutional progression, not the conversational history.
